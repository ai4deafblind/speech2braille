# Client

React web interface for speech-to-braille.

## Setup
```bash
pnpm install
```

## Development
```bash
pnpm dev        # Start dev server
pnpm lint       # Run ESLint
pnpm build      # Production build
```

## Generate API Client
```bash
pnpm generate-client  # Requires server running at :8000
```

## Stack
React 19, Vite, TailwindCSS 4, TanStack Router/Query, Zustand
